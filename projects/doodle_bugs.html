Doodle Bugs
2020 · JavaScript
A puzzle game based around an uncooperative drawing tool.
Doodle Bugs is a web-based puzzle game, playable at www.doodlebugs.art. The central feature is a drawing tool with ten colors, each of which has its own "bug" that users must discover. For example, green strokes gradually erase themselves the more you draw, and purple strokes fall to the bottom of the canvas as if affected by gravity. The game presents a series of images that users must draw, accounting for and making use of the tool's "bugs." Their drawings are auto-graded and accepted if they are close enough to the goal image and don't use too many strokes.


Doodle Bugs' interface. Faded guidelines for the goal image are on the canvas.

Doodle Bugs is an extension of a puzzle called "Stroke of Genius" that was featured in Puzzle Hunt CMU's S20 hunt. I co-wrote "Stroke of Genius" with Tom Wildenhain. The mechanic was inspired by Ben Fry's fugpaint.

Sketch Garden
2020 · JavaScript
A grid of 100 recorded attempts to solve a level of Doodle Bugs.
Sketch Garden is a typology project related to Doodle Bugs. When the game was featured in the CMU puzzle hunt as "Stroke of Genius," over 500 participants played it. I recorded everyone's keystrokes, and arranged 100 attempts at the flower level into a grid organized by general strategy. This format of displaying many drawings in a grid was inspired in part by Aaron Koblin’s Sheep Market. This project was made for the course Experimental Capture, and you can read more about it in this blog post.
